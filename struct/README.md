#  代码功能
- 用自定义 3D 向量工具完成两类几何计算：
  1. 点到直线距离（示例：点 B → 直线 AC₁）
  2. 点到平面距离（示例：点 F → 平面 AEC₁）

#  工具函数速览
| 函数 | 作用 |
| --- | --- |
| `vec_add` | 向量加法 |
| `vec_sub` | 向量减法 |
| `vec_dot` | 点乘（内积）|
| `vec_cross` | 叉乘（外积）|
| `vec_len` | 求模 |
| `vec_normalize` | 单位化 |
